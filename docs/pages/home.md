# Home / Dashboard

The Home page is the first thing you see after selecting a configuration. It gives you a quick snapshot of your platform's activity and health.

<p align="center">
  <img src="../../images/home-dashboard.png" alt="Home dashboard with KPI cards and charts" width="90%">
</p>

## KPI cards

The top row shows key metrics for the active configuration — request counts, error rates, active subscriptions, and other platform stats. Each card displays the current value and a trend indicator.

## Charts

Below the KPI cards you'll find:
- **Activity chart** — area chart showing request volume over time
- **Radial charts** — show distribution across different event types or endpoint groups

Charts are rendered directly from the platform's analytics data and update on page refresh.

## Hero banner

At the top of the page, the banner shows the platform name and the first 8 characters of your active configuration ID so you always know which workspace you're in.

## Activity feed

A scrollable feed of recent platform events — requests processed, agent runs completed, subscription triggers fired. Useful for a quick health check without opening Playground.

## Navigation tip

If the KPI cards show no data, make sure you have an active configuration selected. Click the config badge in the header to go to Settings.
